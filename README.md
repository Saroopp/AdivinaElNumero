## 📖 Descripción

**Adivina el Número** es un programa desarrollado en Python en el que la computadora intenta adivinar el número que el usuario ha pensado. Para lograrlo, el usuario responde si el número propuesto es mayor, menor o correcto, permitiendo que el sistema ajuste el rango de búsqueda hasta encontrar la respuesta.

---

## 🎯 Objetivo

Desarrollar un sistema interactivo que aplique los conocimientos adquiridos durante la asignatura de Lógica de Programación mediante el uso de algoritmos, estructuras condicionales y ciclos para resolver un problema de forma lógica.

---

## ⚙️ Funcionalidades

- Inicia una nueva partida.
- La computadora intenta adivinar el número.
- El usuario responde si el número es mayor, menor o correcto.
- El sistema ajusta automáticamente el rango de búsqueda.
- Muestra el número de intentos realizados.
- Permite jugar nuevamente.
